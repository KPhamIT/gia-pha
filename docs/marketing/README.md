# Kịch bản quảng cáo Facebook — Gia phả điện tử

Bộ nội dung dùng cho fanpage / nhóm Facebook, Zalo OA, hoặc tin nhắn giới thiệu dịch vụ.

## Cách dùng

1. Chọn file bài phù hợp mục tiêu (người xem vs ban quản trị dòng họ).
2. Copy phiên bản **Ngắn** (feed) hoặc **Dài** (storytelling) — chỉnh tên dòng họ / link thật trước khi đăng.
3. Chụp ảnh minh họa theo gợi ý trong từng file (thay placeholder trên `/huong-dan` nếu cần).
4. Gắn link CTA: trang chủ, hướng dẫn, hoặc liên kết join do admin cấp.

## Thay placeholder trước khi đăng

## Quy ước trong từng file

- **Phiên bản A — Ngắn:** 3–6 dòng, phù hợp feed mobile.
- **Phiên bản B — Dài:** kể chuyện, tăng tương tác comment.
- **Carousel / Album:** text từng slide nếu đăng nhiều ảnh.
- **Gợi ý ảnh / video:** shot màn hình cần chụp.
- **Hashtag:** tùy chọn, không lạm dụng.

## Lưu ý pháp lý & nội dung

- Không hứa hẹn quá mức (ví dụ: “100% không bao giờ quên giỗ” — nên nói “nhắc trước”, “hỗ trợ”).
- Push thông báo cần người dùng **bật quyền trình duyệt** và **đăng nhập** — nêu rõ trong bài.
- Bài cúng tự động dựa trên **mẫu** do dòng họ soạn; hệ thống **điền tên, ngày giỗ âm lịch, dòng họ…** — không thay thế hoàn toàn văn hóa / tục lệ địa phương.
